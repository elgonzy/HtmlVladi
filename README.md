### For use docker enviroment.

    1.  Install [Docker-desktop](https://www.docker.com/products/docker-desktop/) in your OS. 
    1.  In the folder after the repository, run this command `` docker build HtmlVladi `` .
    1.  In repository folder run `` docker-compose up ``.
    1.  Now docker is working in this ports: Apache in 8000, Mysql in 9906.
